// Extensions:
This folder contains some most common extionsions that require while swift app development
and helps development easy


// Communication Layer
CommManager class can be used for api calling. This code already setup to use objectiveC like completions handlers.
This class is using Alomafire for api calling. 
Tip: So install Alomafire in ur project using Pods.

// Business Layer
This folder contains class that act as middle man between ur View/ViewControllers and Communication Layer.
This file contain sample code that can be used to create further methods.
You can also use this class to parse your api response to your data models and return them back to ur View/ViewControllers
